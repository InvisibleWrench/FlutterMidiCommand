MidiBleTransport

MidiBleTransport class abstract

BLE transport contract consumed by MidiCommand.

This is intentionally separate from MidiCommandPlatform so native platform wrappers can remain focused on serial/native MIDI stacks while BLE is provided by shared Dart implementations (for example universal_ble).
